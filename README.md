# Explorer v2.1

Aplicativo multiusuário para trilhas, registros de vida selvagem, fotografia, comunidade e segurança em campo.

## Publicação

- Frontend estático: Vercel
- Código-fonte: GitHub
- Banco, autenticação e mídias: Supabase
- Domínio: https://explorer-six-self.vercel.app

## Recursos

- Cadastro aberto para múltiplos usuários
- Trilhas públicas e privadas com gravação GPS
- Observações e fotografias de vida selvagem
- Feed, curtidas, comentários e perfis públicos
- Compartilhamento nativo, WhatsApp, Facebook e links públicos
- Mapa OpenStreetMap
- Modo SOS com ativação e desligamento, sirene, voz, sinal visual, tentativa de lanterna e compartilhamento de localização

## Privacidade

Cada usuário controla a publicação de trilhas e observações. Dados de segurança e conta permanecem isolados por políticas de acesso no Supabase.
